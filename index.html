Irresposible Code

Irresponsible Code — a free-form coding style that optimizes for creativity, exprimentation and fun.

At work, you write maintainable, well designed and fully tested code, responsible code. This is great for production quality code. But it can also be limiting.

Irresponsible Code promotes writing neat hacks, expirementation with new and unusual technologies, and not writing unit tests.

Try writing some Irresponsible Code tonight or this weekend. Have fun and make awesome stuff!
